# HamroNepal_Frontend_project
Hamro Nepal is a basic front-end web project created using HTML and CSS. This project is designed to showcase a simple yet elegant representation of Nepal through a web page.

Project Overview
Hamro Nepal is a static website project that provides a visual and informational representation of Nepal. It is built using only HTML and CSS, demonstrating fundamental web development skills.

Features
Responsive design
Clean and minimalistic layout
Informative content about Nepal
Easy to navigate
